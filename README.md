# AdventOfCode2025

F# solutions to the [Advent of Code 2025](https://adventofcode.com/2025) challenge.

Run from a terminal window:
```cmd
cd Day1
dotnet run
```

I took this opportunity to look at the [FParsec](https://www.quanttec.com/fparsec/) parser combinator library, as well as refresh my knowledge of the [Seq Module](https://fsharp.github.io/fsharp-core-docs/reference/fsharp-collections-seqmodule.html) functions in [FSharp.Core](https://fsharp.github.io/fsharp-core-docs/reference/fsharp-core.html).

> [!NOTE]
> [.NET 10](https://dotnet.microsoft.com/en-us/download/dotnet/10.0) installation is required.
